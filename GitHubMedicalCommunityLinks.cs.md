# GitHub Medical Community Links (CZ)

## Účel

Tento dokument mapuje open-source projekty, které mohou sloužit jako stavební bloky, moduly, výzkumné základy nebo inspirace pro platformu MedBay.

Cílem není sbírat repozitáře podle popularity nebo aktivity, ale identifikovat využitelné medicínské schopnosti.

- moduly
- algoritmy
- datasety
- výzkumné základy
- tréninkové prostředí

MedBay zůstává jednotná modulární medicínská platforma.
